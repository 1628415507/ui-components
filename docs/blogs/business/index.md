## Web ⽹⻚如何禁⽌别⼈移除⽔印
- `MutationObserver` 可以观察 DOM 树的变化，并在变化发⽣时触发回调函数。
- 可以在回调函数中使用`mutation.removedNodes`检查是否有⽔印被删除，
- MutationObserver API 是现代浏览器提供的功能，在⽼旧的浏览器中可能不⽀持。因此，在实际使⽤时，需要注意对浏览器的兼容性进⾏测试和处理。

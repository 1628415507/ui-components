# vitepress 目录说明

```
.
├─ docs                         # 项目根目录
│  ├─ .vitepress                # 配置目录
│  ├─  ├─ assets                # 样式资源文件
│  ├─  ├─ config                # 自定义配置、
│  ├─  ├─ customMdContainer     # 自定义md容器组件
│  ├─  ├─ theme                 # 主题配置
│  ├─  ├─ utils                 # 公共方法
│  ├─  ├─ config.ts             # 公共配置
│  ├─  ├─ menu.ts               # 左侧菜单
│  ├─  ├─ config.mts            # markdown菜单配置
│  └─ components                # 文档中使用的组件
│  └─ examples                  # 组件示例代码、说明文档
└─ ...
```
